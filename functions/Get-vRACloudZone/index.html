    Get-vRACloudZone

    SYNOPSIS

    Get a vRA Cloud Zone

    SYNTAX

    Standard (Default)

    Get-vRACloudZone [<CommonParameters>]

    ById

    Get-vRACloudZone -Id <String[]> [<CommonParameters>]

    ByName

    Get-vRACloudZone -Name <String[]> [<CommonParameters>]

    DESCRIPTION

    Get a vRA Cloud Zone

    EXAMPLES

    EXAMPLE 1

    Get-vRACloudZone

    EXAMPLE 2

    Get-vRACloudZone -Id '3492a6e8-r5d4-1293-b6c4-39037ba693f9'

    EXAMPLE 3

    Get-vRACloudZone -Name 'TestCloudZone'

    PARAMETERS
